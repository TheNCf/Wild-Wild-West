# Wild-Wild-West

## Controls
| Action        | Control                              |
| ------------- | ------------------------------------ |
| Movement      | <kbd>WASD</kbd> or <kbd>Arrows</kbd> |
| Walk (toggle) | <kbd>Alt</kbd>                       |
| Run           | Hold <kbd>Shift</kbd>                |
| Draw weapon   | Hold <kbd>RMB</kbd>                  |
